java programming language

In June 1991, James Gosling launched a Java language framework for use in one of his many experiments on the set-top box.
The language, originally called 'Oak' after an oak tree standing outside the office of Gosling, also went by the name 'Green'
And was later renamed Java, from a compilation of random terms. In 1995 Sun released Java 1.0 as the first public implementation.

Java is ::

  • Object Oriented
  • Portable
  • High Performance
  • Dynamic

Applications on java :

  1. Games
  2. Web applications
  3. Mobile applications (specially Android apps)
  4. Web servers and application servers
  5. And much, much more!
